💻 Why I Started Programming

Not to master every syntax — but to understand how code works and build ideas with AI.

When I first looked at code, I didn’t think I could ever write it. It looked complex, filled with symbols and logic that didn’t make sense at first glance. But curiosity pulled me in. I wanted to understand how things worked — not to become a professional programmer, but to see how technology actually breathes from the inside.

So I began small. I started watching YouTube lectures instead of taking official paid courses. I found Harvard’s CS50 lectures on YouTube — the same ones taught by David J. Malan — and they became my foundation. I didn’t take the course from the official site, but I studied every topic from those lectures at my own pace.

“I didn’t learn programming to memorize syntax — I learned it to understand what happens when I run a line of code.”

Learning from YouTube and mentors

Along with CS50, I learned a lot from CodeWithHarry — his tutorials made concepts simple and practical. From him, I learned Python and C in depth, and understood how programming logic actually flows. I also used FreeCodeCamp to strengthen my basics and practice writing clean code.

FreeCodeCamp — YouTube CodeWithHarry — Python & C Harvard CS50 — YouTube Lectures

Most of my learning happened through videos, trial, and error. I’d watch a concept, try it myself, make mistakes, and ask AI tools like ChatGPT for help when stuck. That combination — YouTube + AI + self-practice — became my personal classroom.

Programming with AI

Over time, I realized I didn’t need to know every language deeply — I just needed to understand the logic behind it. Now, I mostly use AI tools to help me build and debug my projects. I read the code, try to understand what it’s doing, and improve it through feedback and learning. I don’t code everything from scratch, but I make sure I understand what’s happening behind every project I create.

How programming connects with my journey

Programming gave me the confidence to build ideas like My Mitra and IntervAI. It helped me bridge the gap between imagination and creation — even when coding felt difficult. Every project I built taught me more about logic, structure, and the power of combining AI with programming.

“For me, programming isn’t about typing — it’s about understanding, creating, and connecting ideas with reality.”

I still learn programming every day — not from textbooks, but from doing, watching, and experimenting. My goal isn’t to be a perfect coder, but to be a creator who understands how things work and can use AI to build something meaningful.
